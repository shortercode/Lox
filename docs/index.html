What is Lox?

Lox is a programming language devised by Robert Nystrom for the book "Crafting Interpreters". It's dynamically typed, classy and I think pretty neat. Here's a little snippet:

          class Breakfast {
            init(meat, bread) {
              this.meat = meat;
              this.bread = bread;
            }

            // ...
          }
          class Brunch < Breakfast {
            init(meat, bread, drink) {
              super.init(meat, bread);
              this.drink = drink;
            }
          }
          var benedict = Brunch("ham", "English muffin");
          benedict.serve("Noble Reader");
